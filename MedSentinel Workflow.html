AI Platform for Affordable Healthcare & Government Support
Complete User Workflow Guide
1
Landing Page & Registration

User Action: Patient/family member visits MedSentinel platform

📝 Initial Input Required:
Account Creation (Optional) - Email, Password for saving searches
Guest Mode Skip registration, proceed directly to search
🎯 What User Sees:
  • Welcome message explaining platform purpose
  • Quick overview of features (compare costs, find schemes, get recommendations)
  • Call-to-action button: "Find Affordable Healthcare"
↓
2
Medical Condition Search

User Action: Enter medical condition or treatment needed

📝 Required Input:
Medical Condition/Procedure e.g., "Heart Surgery", "Diabetes Treatment", "Cancer Chemotherapy", "Kidney Stone Removal"
Location Preference City or State (e.g., Bengaluru, Karnataka)
Treatment Urgency Emergency / Scheduled / Flexible
🎯 What User Sees:
  • Auto-suggest dropdown with common medical conditions
  • Search bar with AI-powered natural language understanding
  • Location selector with nearby cities
  • Loading indicator while AI processes request
↓
3
Patient Profile Details

User Action: Provide personal information for government scheme eligibility check

📝 Required Input:
Full Name Patient's legal name
Age Patient's current age
Gender Male / Female / Other
Income Category Below Poverty Line (BPL) / Above Poverty Line (APL) / Annual Income Range
State of Residence For state-specific schemes
Aadhaar Number (Optional but recommended) - For Ayushman Bharat eligibility
Existing Health Card Ayushman Bharat card number if already enrolled
🎯 What User Sees:
  • Form with clear field labels and validation
  • Privacy assurance message
  • Progress indicator showing "Step 2 of 3"
  • Option to skip optional fields
↓
4
AI Analysis & Processing

System Action: AI agent processes data in the background

🤖 Backend Operations:
  • Data Collection: Fetch treatment costs from healthcare databases and hospital APIs
  • Price Analysis: AI compares costs across multiple hospitals for the specified treatment
  • Scheme Matching: AI checks eligibility for government healthcare schemes based on patient profile
  • Recommendation Generation: GPT-4 or Claude AI generates personalized recommendations
🎯 What User Sees:
  • Loading screen with progress messages ("Analyzing treatment costs...", "Checking scheme eligibility...", "Generating recommendations...")
  • Estimated wait time: 10-15 seconds
↓
5
Results Dashboard

System Output: Comprehensive results displayed in organized sections

📊 Section 1: Hospital Cost Comparison
🏥
List of Hospitals - Sorted by cost (lowest to highest)
  • Hospital Name & Address
  • Estimated Treatment Cost (with breakdown)
  • Distance from patient location
  • Hospital rating & patient reviews
  • Available specialties & facilities
💰
Cost Breakdown - Transparent pricing
  • Consultation fees
  • Procedure/surgery costs
  • Medication expenses
  • Hospital stay charges
  • Additional tests/diagnostics
🎁 Section 2: Government Scheme Eligibility
✅
Eligible Schemes - Programs patient qualifies for
  • Ayushman Bharat (PM-JAY): Coverage amount, eligibility status, enrollment status
  • State Health Schemes: Karnataka-specific programs like Vajpayee Arogya Shree, Yeshasvini
  • Central Government Schemes: CGHS, ESI, ECHS (if applicable)
  • Financial Support Amount: How much coverage/subsidy available
📋
How to Apply - Step-by-step guidance
  • Documents required for enrollment
  • Application process (online/offline)
  • Official website links
  • Helpline numbers
  • Application timeline
⭐ Section 3: AI Recommendations
🎯
Top Recommended Option
  • Most affordable hospital + government scheme combination
  • Estimated out-of-pocket cost after subsidies
  • Why this option is recommended (AI explanation)
  • Potential savings compared to average cost
📍
Alternative Options
  • 2-3 backup hospital recommendations
  • Quality vs cost trade-offs explained
  • Special considerations (emergency facilities, specialist availability)
📄 Section 4: Next Steps
  • Download Report: PDF with complete cost comparison & scheme details
  • Contact Hospitals: Direct call/email buttons for shortlisted hospitals
  • Apply for Schemes: Links to government portals with pre-filled application data
  • Save for Later: Bookmark results to registered account
  • Share Results: Send via email/WhatsApp to family members
↓

Technical Implementation Flow
Backend Workflow:
1. User Input → FastAPI Backend
Next.js frontend sends request to FastAPI server
2. LangChain Agent Orchestration
LangChain coordinates multiple AI tasks (data retrieval, analysis, recommendation)
3. Data Sources Query
Fetch data from healthcare datasets, NHA portals, India API Setu
4. GPT-4/Claude Analysis
AI model processes data and generates insights
5. Database Storage
Store results in MongoDB/PostgreSQL for user retrieval
6. Response to Frontend
Send formatted results to Next.js for display
Tech Stack:
Frontend
Next.js
Backend
FastAPI + Python
AI Layer
LangChain + GPT-4/Claude
Database
MongoDB or PostgreSQL
